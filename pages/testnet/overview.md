# StableNet Testnet Overview

## What is StableNet Testnet?
StableNet Testnet is a testing environment for developers to experiment with StableNet features without using real assets.

## Who is this for?
- Developers with Web2 background
- Developers new to Web3

## Development Flow
1. Prepare a wallet
2. Request test tokens from the Faucet
3. Connect to StableNet Testnet RPC
4. Send your first transaction

## Important Notes
- Testnet data may be reset at any time
- Testnet tokens have no real value
